# SnowLog

SnowLog は、スキー・スノーボード動画をかんたんに記録・整理できるアプリです。
撮ったまま埋もれがちな一本を、あとから気持ちよく振り返れるようにします。

## SnowLog でできること

- ゲレンデ名やメモを添えて、滑走動画を自分だけのログとして残せます。
- 技術やタグで整理できるので、練習内容や成長の流れを見返しやすくなります。
- お気に入り登録で、特に残しておきたい一本をすぐ見つけられます。
- カレンダーから日ごとの記録をたどれて、滑った日々を一覧で振り返れます。
- 統計表示で、どれくらい滑ったか・記録したかを感覚的につかめます。

## こんな人におすすめ

- スキーやスノーボードの動画をあとで見返したい人
- トリックやフォームの練習記録を残したい人
- ゲレンデごとの思い出をきれいに整理したい人
- 写真フォルダの中に動画が埋もれて困っている人

## 使い方はシンプル

1. 滑走動画を取り込む
2. ゲレンデ名やタグ、メモをつける
3. お気に入りやカレンダーから振り返る
4. 自分の滑りの記録を少しずつ育てていく

## 目指している体験

SnowLog は、ただ保存するだけでなく、
「あの日どこで滑って、どんな一本を残したか」を気持ちよく思い出せる体験を大切にしています。
練習の記録にも、仲間との思い出の整理にも使える、雪山好きのためのログアプリです。
